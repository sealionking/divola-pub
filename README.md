# DiVo Learning Assistant (Windows Portable)

这是 DiVo 学习助理的 Windows 便携版发行仓库。

### 🌐 在线试用
您可以直接访问在线版本进行试用：[https://divola.wangjueju.cn](https://divola.wangjueju.cn)

### 📥 下载说明
请前往 [Releases](https://github.com/sealionking/divola-pub/releases) 页面下载最新的 `.exe` 单文件版。下载后双击即可直接运行，无需安装。

### 🛡️ 安全校验
每个发布版本都附带了 SHA-256 校验码，建议在运行前进行比对。
